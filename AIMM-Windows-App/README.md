# AIMM Windows App

Open `Launch AIMM App.bat` or `index.html` on a Windows laptop.

The app stores draft assessments in browser local storage and supports:

- 5-pillar AIMM scoring: Data maturity, AI infra, Cyber posture, Process & BizOps, Talent & culture
- 0-5 maturity scoring aligned to the source framework deck
- Question-level evidence notes under each pillar
- Overall readiness score, maturity level, heatmap, and recommended next phase
- Agentic AI / GenAI impact-vs-effort use case prioritisation
- JSON import/export and board-ready HTML report export

Primary local source files used:

- `01 AIMM/AI-Readiness & Agentic AI Adoption framework.pptx`
- `01 AIMM/Enterprise AI Maturity Framework.pdf`
- `01 AIMM/Practical_Lessons_AI_Agents.pdf`
- `01 AIMM/LLM Agent Evaluations & Project Overview.pdf`
- Other AIMM agentic model, evaluation, and infrastructure PDFs in `01 AIMM`

External reference basis embedded in the app:

- NIST AI Risk Management Framework
- NIST Cybersecurity Framework 2.0
- ISO/IEC 42001:2023
- OWASP Top 10 for LLM Applications
- Microsoft AI Readiness Assessment framing
